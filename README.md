# QR Code Generator
A simple QR code generator for MacOS and Windows.
Prequisities:
Must have Python 3.6 or higher pre-installed

# Pre-Setup Instructions (MUST PERFORM OR THE APP WON'T WORK)
1. Windows: Open the Start Menu and search "Command prompt"
1. Mac: Press CMD+Space and type "Terminal"
3. Click on the first app that appears
4. In the app window, type "python -m pip install qrcode" and press Enter
5. A bunch of text should roll across the screen. Do not touch anything
6. When you can type again, type "python -m pip install pillow" and press Enter
7. Once again, a bunch of text should roll across the screen. When you can type again, type "exit"
8. You're all set to install the app!

# Windows Setup Instructions
1. Download the Windows setup file from the releases page of this GitHub repository
2. Click on the setup file
3. The wizard will walk you through the setup steps
4. Once you have finished setting the app up, launch it
5. On the blue window that pops up, click "More info"
6. Click "Run anyway"

# MacOS Setup Instructions (FAULTY ON MAC)
1. Download the MacOS setup file from the releases page of this GitHub repository
2. Click on the downloaded setup file
3. Drag the app icon to the Applications folder
4. Press CMD+Space and search "QR Code Generator"
5. Click on the app
6. If the OS refuses to let you run the app, defer to YouTube for instructions on how to bypass this restriction

# App Usage Instructions
1. Launch the app
2. In the top box, type in the URL of the site you want to make a QR code for
3. In the second box, type in the path to the folder where you want the QR code file to be stored
4. In the third box, type in the name of your QR code file
5. Click "Generate"
6. Accept the popup and launch your device's file manager
7. Navigate to the folder you selected and double-click on the PNG file
8. Enjoy!
